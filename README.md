# telegram-calorie-bot

Бот для отслеживания воды, калорий и тренировок.

## Функционал
- Расчет нормы воды и калорий
- Логирование питания через OpenFoodFacts API
- Учет погоды через OpenWeatherMap API
- Трекинг тренировок
- Графики прогресса
- Персональные рекомендации

## Команды
- /setup - настройка профиля
- /drink - добавить воду
- /eat - добавить еду
- /train - добавить тренировку
- /status - прогресс
- /charts - графики
- /tips - рекомендации
